### 寻找两个有序数组的中位数
* 思路一
    * 依旧从最简单粗暴的方法开始, 既然是两个数组的中位数, 那么很自然而然就会想到, 先把两个数组合并, 然后进行排序, 然后直接用数组的下标去取中位数(当然这里要考虑一下奇偶的情况), 就可以了
    * 因为我使用了js原生的sort api , 底层的v8引擎在实现时使用了最优的排序, 了解到在小样本时使用的是插入排序, 大样本时使用的是快排, 所以这里实际上是偷懒了
